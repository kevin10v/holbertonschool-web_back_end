# ES6 Data Manipulation

This project focuses on ES6 data manipulation techniques including:
- Using map, filter, and reduce on arrays
- Typed arrays
- Set, Map, and WeakMap data structures

## Requirements
- Ubuntu 20.04 LTS
- Node 20.x.x
- npm 9.x.x

## Setup
```bash
npm install
```

## Usage
Run individual files:
```bash
npm run dev <filename>
```

Run tests:
```bash
npm test
```

Run full test (lint + tests):
```bash
npm run full-test
```
