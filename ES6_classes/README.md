# ES6 Classes

This project focuses on ES6 classes, including:
- How to define a Class
- How to add methods to a class
- Why and how to add a static method to a class
- How to extend a class from another
- Metaprogramming and symbols

## Requirements

- All files interpreted/compiled on Ubuntu 20.04 LTS using node 20.x.x and npm 9.x.x
- All files should end with a new line
- Code uses the `.js` extension
- Code tested using Jest: `npm run test`
- Code verified against lint using ESLint
- Code needs to pass all tests and lint: `npm run full-test`

## Setup

Install dependencies:
```bash
npm install
```
